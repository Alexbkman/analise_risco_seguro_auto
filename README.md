# Análise de Risco - Seguro Auto

## Objetivo
Analisar fatores associados à ocorrência de sinistros em seguros de automóveis.

## Dataset
Car Insurance Claim Dataset (Kaggle)

## Etapas
- Auditoria e tratamento de dados
- Análise exploratória (EDA)
- Identificação de drivers de risco
- Geração de insights e recomendações

## Principais insights
- Score de crédito baixo está associado a maior risco
- Histórico de acidentes aumenta significativamente a probabilidade de sinistro
- Menor experiência de direção está correlacionada com maior incidência de sinistros

## Ferramentas
- Python (Pandas, Seaborn)
- Power BI (para dashboard)
